# Installs latest version of [Ammonite-REPL](https://ammonite.io/#Ammonite-REPL) on [Ubuntu](https://www.ubuntu.com/)

```bash
wget -q https://script.install.devinsideyou.com/ammonite-repl
chmod +x ammonite-repl && ./ammonite-repl
```
